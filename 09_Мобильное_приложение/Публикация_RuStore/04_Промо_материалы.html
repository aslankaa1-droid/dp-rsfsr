3. Скриншоты

Минимальные требования RuStore

  • Минимум 5 скриншотов, максимум 8.
  • Соотношение сторон: 16:9 (горизонтальный) или 9:16 (вертикальный).
  • Минимальный размер: 320 × 480.
  • Рекомендуемый размер: 1080 × 1920 (вертикальный).
  • Формат: PNG, JPEG.

4. Промо-видео

Параметр Значение
Длительность 15–30 секунд
Формат MP4 (H.264 + AAC)
Разрешение 1080 × 1920 (вертикальное)
Аудио с закадровым голосом или субтитрами

Сценарий (15 секунд): - 0–3 с: логотип + название. - 3–6 с: главная — баланс и активы. - 6–9 с: подача актива. - 9–12 с: подписание смарт-контракта. - 12–15 с: «ЦФА выпущен» + слоган.

Инструкция по подготовке скриншотов

  1. Откройте Приложение в браузере на телефонной симуляции (DevTools → Mobile, разрешение 1080 × 1920).
  2. Пройдите соответствующий экран.
  3. Сделайте скриншот через DevTools (Cmd/Ctrl + Shift + P → «Capture screenshot»).
  4. При необходимости — добавьте подпись и оформление в графическом редакторе.

Команда генерации иконок (требует ImageMagick или Pillow)

# С помощью Pillow / Python
python -c "
from PIL import Image
import cairosvg, io

for size in [44, 71, 96, 150, 192, 310, 512]:
    cairosvg.svg2png(url='Приложение/icons/icon-512.svg', write_to=f'store-{size}.png', output_width=size, output_height=size)
"

Альтернативно — @capacitor/assets CLI генерирует все размеры автоматически.
