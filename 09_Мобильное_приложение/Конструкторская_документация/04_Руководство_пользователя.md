# Руководство пользователя

## ЦП РСФСР — мобильное приложение

**Условное обозначение:** РСФСР.МП.04.РП
**Стандарт:** ГОСТ 19.505-79

---

## 1. Назначение программы

Приложение предназначено для управления личным портфелем активов и цифровых финансовых активов (ЦФА) на Цифровой Платформе РСФСР: подача имущества на залог, выпуск ЦФА, получение дохода от размещения, контроль смарт-контрактов.

## 2. Условия выполнения программы

### 2.1. Минимальные технические требования
- iOS 15 и выше, либо
- Android 9 (API 28) и выше, либо
- Windows 10 (1809) и выше, либо
- современный браузер с поддержкой Service Worker.

### 2.2. Подготовка к работе
1. Установите приложение из RuStore, App Store, Google Play, Microsoft Store, либо добавьте PWA на главный экран из браузера.
2. Подготовьте подтверждённую учётную запись Госуслуг.
3. Подготовьте документы на актив (правоустанавливающие, кадастровый номер для недвижимости).

## 3. Работа с программой

### 3.1. Первый запуск

При первом запуске приложение приветствует пользователя экраном-заставкой и автоматически переходит на экран входа.

#### Вход в систему
Доступны два способа:
- **Войти через Госуслуги** — основной способ для новых пользователей. Используется OpenID Connect к ЕСИА с подтверждённой учётной записью.
- **Биометрия** — для повторных входов уже зарегистрированного пользователя через Face ID / Touch ID / Windows Hello.

#### Регистрация (KYC)
После первичной авторизации проводится трёхшаговая проверка личности:
1. **Подтверждение личности**: Фамилия, Имя, Отчество, серия и номер паспорта, ИНН. Данные подгружаются из подтверждённой учётной записи Госуслуг — проверьте их корректность.
2. **Контактные данные**: телефон (для SMS-кодов), email, регион.
3. **Биометрия**: подтверждение через Единую биометрическую систему. Поднесите телефон к лицу или приложите палец к датчику отпечатков. Можно пропустить этот шаг и настроить биометрию позже из меню «Настройки».

#### Знакомство с возможностями (Onboarding)
После регистрации показываются три обучающих экрана:
1. Внесите актив в залог — недвижимость, транспорт, ИС, ценные бумаги; имущество остаётся у вас.
2. Платформа выпускает ЦФА — под ваш актив; Банк России производит целевую эмиссию.
3. Получайте доход — регулярные выплаты; по истечении срока обременение снимается.

Любой шаг можно пропустить, нажав «Пропустить» в правом верхнем углу.

### 3.2. Главный экран

Главный экран содержит:
- **Карточка баланса** — общая стоимость активов в работе; количество объектов в залоге; доход за квартал.
- **Быстрые действия**: Подать актив, Мои ЦФА, Доходы, Документы.
- **Баннер национального проекта** — новые программы финансирования.
- **Мои активы** — два последних актива с возможностью открыть полный список.
- **Последние операции** — две последние операции с возможностью открыть полную историю.
- **Нижняя навигация** — пять вкладок: Главная, Активы, ЦФА, Доходы, Профиль.

### 3.3. Подача актива на залог

#### Шаг 1. Выбор типа имущества
Доступные категории:
- Недвижимость (квартира, дом, апартаменты);
- Земельный участок;
- Транспортное средство;
- Интеллектуальная собственность;
- Ценные бумаги.

#### Шаг 2. Характеристики
Для недвижимости — введите кадастровый номер. Адрес, площадь, год постройки и другие характеристики будут подгружены автоматически из ЕГРН через защищённый канал интеграции с Росреестром (218-ФЗ). Также прикрепляется документ о праве собственности.

#### Шаг 3. Заявка на оценку
Просмотрите параметры заявки и подтвердите отправку. Оценочная компания-партнёр, аккредитованная по 135-ФЗ, проведёт оценку в течение 24 часов. Стоимость заявки — 0 ₽ (включено в платформенный сервис).

#### Шаг 4. Результат оценки
Получите оценочную стоимость и максимальный размер залогового ЦФА (по умолчанию 50% от оценки). Страхование залога включено. Действует 6 месяцев. Перейдите к смарт-контракту, чтобы выпустить ЦФА.

### 3.4. Настройка и подписание смарт-контракта

На экране смарт-контракта установите:
- **Срок ЦФА**: 12, 36 или 60 месяцев.
- **Целевое использование**: программа национального или регионального проекта.
- **Номинал ЦФА**: рассчитывается автоматически как 50% от оценочной стоимости.

Просмотрите расчётные параметры: ставка вознаграждения, доход за весь срок, эскроу-агент (Банк России), страхование, регистратор обременения (Росреестр).

Нажмите «Подписать (УКЭП)». На вашу телефонную линию придёт SMS-код. Введите 6-значный код или подтвердите биометрией.

После успешного подписания вы увидите экран «ЦФА успешно выпущен» с уникальным идентификатором ЦФА. ЦФА передан в Банк России, обременение зарегистрировано в ЕГРН.

### 3.5. Управление портфелем ЦФА

В разделе «Мои ЦФА» доступен полный список ваших ЦФА с фильтрацией: Все, Активные, На выпуске, Завершённые. По каждому ЦФА показаны:
- ID (уникальный идентификатор);
- Номинал;
- Связанный актив;
- Целевая программа;
- Срок и ставка вознаграждения;
- Статус.

При клике на ЦФА открывается карточка с детальной информацией: условия смарт-контракта, эскроу, выплаты, история. Доступны действия: скачать договор, запросить мену ЦФА.

### 3.6. Доходы

В разделе «Доходы»:
- Карточка баланса доходов: за год, за квартал, прогноз годовой.
- График помесячных поступлений.
- Список выплат с привязкой к ЦФА.

### 3.7. История операций

Полный журнал операций с фильтрацией: Все, Выплаты, Выпуск ЦФА, Мена. Можно экспортировать в PDF или Excel.

### 3.8. Уведомления

Центр уведомлений содержит:
- Системные сообщения о выплатах;
- Уведомления о новых программах;
- Уведомления о результатах оценок;
- Уведомления о регистрации обременений и других событиях смарт-контракта.

Непрочитанные уведомления помечены точкой акцента. По нажатию на колокольчик можно перейти ко всем уведомлениям. По нажатию на иконку «⊘» — все уведомления отмечаются прочитанными.

### 3.9. Профиль

В разделе «Профиль» доступны:
- Личные данные.
- Реквизиты для выплат.
- Мои документы.
- Уведомления.
- Документы платформы.
- Помощь и поддержка.
- Выход из аккаунта.

### 3.10. Настройки

Доступны:
- **Язык**: русский (по умолчанию), английский, французский, арабский.
- **Тема оформления**: светлая, тёмная, мягкая, контрастная для слабовидящих.
- **Размер шрифта**: уменьшить, обычный, увеличить.
- **Безопасность**: биометрия, PIN-код, двухфакторная аутентификация.
- **Уведомления**: push-уведомления.

### 3.11. Документы платформы

Раздел содержит ссылки на полную документацию проекта ЦП РСФСР: хаб документов, большая презентация, малая презентация, веб-сайт, юридический фундамент, международная оценка, DD-аудит, экспертное заключение.

### 3.12. Поддержка

Доступны: FAQ, чат с оператором, телефон поддержки, форма обратной связи.

## 4. Сообщения оператору

### 4.1. Виды сообщений
- Push-уведомления — отображаются на главном экране устройства.
- In-app уведомления — отображаются в центре уведомлений.
- Сообщения от Госуслуг — отображаются в стандартном сервисе Госуслуг.

### 4.2. Действия по сообщениям
Каждое уведомление содержит призыв к действию (например, «Подтвердить выплату», «Запустить выпуск ЦФА»). Перейдите по уведомлению, чтобы открыть соответствующий экран.

## 5. Сообщения об ошибках

| Код | Сообщение | Действие пользователя |
|---|---|---|
| AUTH-01 | Не удалось подтвердить личность через Госуслуги | Повторите попытку, проверьте доступ к Госуслугам |
| AUTH-02 | Биометрия не подтверждена | Введите код подтверждения или подтвердите через ЕБС |
| ASSET-01 | Объект не найден в ЕГРН | Проверьте корректность кадастрового номера |
| ASSET-02 | Объект уже находится в залоге у другой организации | Снимите предыдущее обременение |
| CFA-01 | Не удалось выпустить ЦФА | Свяжитесь с поддержкой |
| NET-01 | Нет соединения с сервером | Проверьте интернет; данные восстановятся после восстановления связи |
| SIGN-01 | Электронная подпись отклонена | Проверьте срок действия сертификата |
