# Руководство по установке и администрированию

## ЦП РСФСР — мобильное приложение

**Условное обозначение:** РСФСР.МП.05.РУ
**Стандарт:** ГОСТ 19.503-79

---

## 1. Общие сведения о программе

Приложение поставляется в формате установочных пакетов под каждую целевую платформу. Дополнительно доступно в режиме PWA — установка из браузера без посредничества стора.

| Платформа | Пакет | Размер |
|---|---|---|
| Android | `.apk` / `.aab` | до 30 МБ |
| iOS | `.ipa` | до 35 МБ |
| Windows | `.msix` / `.appx` | до 40 МБ |
| PWA | `manifest.json` + `index.html` + `sw.js` | до 1 МБ initial download |

## 2. Установка программы

### 2.1. Установка из RuStore (Android, рекомендуемо для РФ)

1. Откройте RuStore на устройстве.
2. В поиске введите «ЦП РСФСР» или «Российский Сервис Финансирования».
3. Нажмите «Установить».
4. Дождитесь загрузки и установки (~30 МБ).
5. Откройте приложение по иконке на рабочем столе.

#### Минимальные требования
- Android 9 (API 28) и выше.
- 200 МБ свободной памяти.

### 2.2. Установка из App Store (iOS)

1. Откройте App Store.
2. В поиске введите «ЦП РСФСР».
3. Нажмите «Загрузить».
4. Подтвердите Touch ID / Face ID / паролем Apple ID.
5. Дождитесь установки.

#### Минимальные требования
- iOS 15 и выше.
- 200 МБ свободной памяти.

### 2.3. Установка из Google Play (Android)

Аналогично RuStore. Доступно при наличии региональных регуляторных одобрений.

### 2.4. Установка из Microsoft Store (Windows)

1. Откройте Microsoft Store.
2. В поиске введите «ЦП РСФСР».
3. Нажмите «Получить».
4. Дождитесь установки.

#### Минимальные требования
- Windows 10 (1809) или Windows 11.
- 500 МБ свободной памяти.
- Архитектура: x64, ARM64.

### 2.5. Установка как PWA (любая платформа)

PWA-режим работает из любого современного браузера без посредничества стора.

#### Установка на Android (Chrome, Edge):
1. Откройте сайт приложения в браузере.
2. В меню браузера выберите «Установить приложение» или появится баннер установки.
3. Подтвердите установку.
4. Иконка добавится на главный экран.

#### Установка на iOS (Safari):
1. Откройте сайт приложения в Safari.
2. Нажмите «Поделиться».
3. Выберите «На экран Домой».
4. Подтвердите.

#### Установка на Windows (Edge, Chrome):
1. Откройте сайт приложения в браузере.
2. В адресной строке появится иконка установки.
3. Нажмите её и подтвердите.

## 3. Настройка программы

### 3.1. Первый запуск
При первом запуске приложение запрашивает:
- разрешение на push-уведомления;
- разрешение на использование биометрии (опционально);
- авторизацию через Госуслуги.

### 3.2. Параметры конфигурации
Большинство параметров конфигурации настраивается автоматически из манифеста приложения. Пользовательские настройки доступны в разделе «Профиль → Настройки»:
- Язык интерфейса.
- Тема оформления.
- Размер шрифта.
- Биометрия.
- PIN-код.
- Двухфакторная аутентификация.
- Push-уведомления.

### 3.3. Хранение данных
Чувствительные данные хранятся исключительно в защищённых хранилищах операционной системы:
- iOS Keychain.
- Android Keystore.
- Windows Credential Locker.

Кеш данных приложения хранится в зашифрованном виде в директории приложения. Размер кеша ограничен 200 МБ; при превышении старые данные автоматически удаляются.

## 4. Обновление программы

### 4.1. Автоматические обновления
Через стандартный механизм магазина приложений (RuStore, App Store, Google Play, Microsoft Store) с уведомлением пользователя.

### 4.2. Принудительные обновления
Для критичных уязвимостей применяется механизм in-app force-update: при запуске приложения проверяется минимально допустимая версия; при её несоответствии приложение блокирует работу до обновления.

### 4.3. PWA-обновления
Service Worker автоматически проверяет обновления в фоне и при следующем запуске активирует новую версию. Пользователь может принудительно обновить через «Профиль → Настройки → Обновить приложение».

## 5. Удаление программы

### 5.1. С устройства
Стандартное удаление через интерфейс операционной системы:
- Android: долгий тап на иконке → «Удалить».
- iOS: долгий тап на иконке → «Удалить приложение».
- Windows: Параметры → Приложения → ЦП РСФСР → Удалить.
- PWA: правый клик на иконке → «Удалить».

### 5.2. Удаление данных
При удалении приложения автоматически удаляются:
- кеш на устройстве;
- сохранённые секреты в защищённом хранилище;
- настройки приложения.

Серверные данные пользователя удаляются только по официальному запросу через раздел «Профиль → Безопасность → Удалить аккаунт» либо через службу поддержки в соответствии с требованиями 152-ФЗ.

## 6. Администрирование

### 6.1. Распространение в корпоративной среде
Для государственных и корпоративных пользователей доступны:
- MDM-распространение через корпоративный портал.
- Apple Business Manager / Google Workspace Endpoint Management.
- Intune (для Windows).

### 6.2. Мониторинг
Серверная часть платформы фиксирует все обращения приложения. Метрики доступны через регуляторную отчётность.

### 6.3. Обращения в поддержку
- Через раздел «Профиль → Помощь и поддержка» в приложении.
- По телефону горячей линии (указывается в приложении).
- Через форму обратной связи в большой презентации проекта.

## 7. Обеспечение безопасности

### 7.1. Защита от компрометации устройства
При обнаружении jailbreak/root приложение:
1. Выводит предупреждение о небезопасной среде.
2. Блокирует функции выпуска ЦФА и подписания смарт-контрактов.
3. Сохраняет только чтение для уже выпущенных ЦФА.

### 7.2. Защита от потери устройства
При потере устройства пользователь должен:
1. Войти в личный кабинет с другого устройства через PWA или веб-сайт.
2. Перейти в «Профиль → Настройки → Безопасность».
3. Нажать «Заблокировать все сессии».

### 7.3. Восстановление доступа
Восстановление пароля и доступа осуществляется через Госуслуги по стандартной процедуре восстановления учётной записи ЕСИА.
