2. Feature graphic / لافتة

الغرض الحجم الوصف
Feature graphic 1024 × 500 بكسل اللافتة الرئيسية للكتالوج؛ تتضمن الشعار والاسم والعبارة الرئيسية

3. لقطات الشاشة

الحد الأدنى من متطلبات RuStore

  • 5 لقطات شاشة كحد أدنى، و8 كحد أقصى.
  • نسبة الأبعاد: 16:9 (أفقي) أو 9:16 (عمودي).
  • الحد الأدنى للحجم: 320 × 480.
  • الحجم الموصى به: 1080 × 1920 (عمودي).
  • الصيغة: PNG، JPEG.

4. فيديو ترويجي

المعلَمة القيمة
المدة 15–30 ثانية
الصيغة MP4 (H.264 + AAC)
الدقة 1080 × 1920 (عمودي)
الصوت مع تعليق صوتي أو ترجمة

السيناريو (15 ثانية): - 0–3 ثانية: الشعار + الاسم. - 3–6 ثانية: الرئيسية — الرصيد والأصول. - 6–9 ثانية: تقديم الأصل. - 9–12 ثانية: توقيع العقد الذكي. - 12–15 ثانية: «تم إصدار DFA» + الشعار.

5. النصوص

الوصف الموجز (80 حرفاً)

«منصة إصدار DFA مقابل ضمان الممتلكات — خدمة إعلامية وتوضيحية»

الوصف الكامل (انظر الملف 01_Описание_приложения.md)

ما الجديد

«الإصدار الأول. دورة حياة كاملة لإصدار DFA: من تقديم الأصل إلى استلام الدخل.»

الفئة العمرية

18+ (تطبيق مالي يستلزم حساب Gosuslugi مؤكَّداً).

الفئات

  • الأساسية: المالية.
  • الإضافية: الأعمال.

الكلمات المفتاحية

DFA، رهن، عقارات، استثمار، روبل رقمي، بنك روسيا، مشاريع وطنية

تعليمات إعداد لقطات الشاشة

  1. افتح التطبيق في المتصفح بوضع محاكاة الهاتف (DevTools → Mobile، الدقة 1080 × 1920).
  2. اجتز الشاشة المعنية.
  3. التقط لقطة الشاشة عبر DevTools (Cmd/Ctrl + Shift + P → «Capture screenshot»).
  4. عند الحاجة — أضف التعليق والتنسيق في محرر رسومي.

أمر توليد الأيقونات (يستلزم ImageMagick أو Pillow)

# باستخدام Pillow / Python
python -c "
from PIL import Image
import cairosvg, io

for size in [44, 71, 96, 150, 192, 310, 512]:
    cairosvg.svg2png(url='Приложение/icons/icon-512.svg', write_to=f'store-{size}.png', output_width=size, output_height=size)
"

بديلاً — تقوم CLI @capacitor/assets بتوليد جميع الأحجام تلقائياً.
