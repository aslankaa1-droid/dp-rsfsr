Manuel de l'utilisateur

Автор: Кагиров А.-Х.А. · Свидетельство о депонировании №4011265 от 19.12.2024 · Center Group Company · Engagement: CPFSR-RF-DESIGN-2026

Manuel de l’utilisateur

CPFSR — application mobile

Identifiant : РСФСР.МП.04.РП Norme : GOST 19.505-79

1. Objet du programme

L’application est destinée à la gestion du portefeuille personnel d’actifs et d’actifs financiers numériques (AFN) sur la Plateforme numérique CPFSR : soumission de biens en garantie, émission d’AFN, perception des revenus du placement, contrôle des contrats intelligents.

2. Conditions d’exécution du programme

2.1. Exigences techniques minimales

  • iOS 15 et supérieur, ou
  • Android 9 (API 28) et supérieur, ou
  • Windows 10 (1809) et supérieur, ou
  • un navigateur moderne avec prise en charge du Service Worker.

2.2. Préparation au travail

  1. Installez l’application depuis RuStore, App Store, Google Play, Microsoft Store, ou ajoutez la PWA à l’écran d’accueil depuis le navigateur.
  2. Préparez un compte Gosuslugi confirmé.
  3. Préparez les documents de l’actif (titres de propriété, numéro cadastral pour l’immobilier).

3. Travail avec le programme

3.1. Premier lancement

Au premier lancement, l’application accueille l’utilisateur par un écran de splash et passe automatiquement à l’écran de connexion.

Connexion

Deux options sont disponibles : - Se connecter via Gosuslugi — méthode principale pour les nouveaux utilisateurs. OpenID Connect vers ESIA avec un compte confirmé est utilisé. - Biométrie — pour les connexions répétées d’un utilisateur déjà enregistré via Face ID / Touch ID / Windows Hello.

Inscription (KYC)

Après l’autorisation initiale, une vérification d’identité en trois étapes est effectuée : 1. Confirmation de l’identité : nom, prénom, patronyme, série et numéro du passeport, INN. Les données sont chargées depuis le compte Gosuslugi confirmé — vérifiez leur exactitude. 2. Coordonnées : téléphone (pour les codes SMS), e-mail, région. 3. Biométrie : confirmation via le Système biométrique unifié. Présentez le téléphone à votre visage ou posez un doigt sur le capteur d’empreintes. Vous pouvez ignorer cette étape et configurer la biométrie plus tard depuis le menu « Paramètres ».

Découverte des fonctionnalités (Onboarding)

Après l’inscription, trois écrans d’apprentissage s’affichent : 1. Mettez un actif en garantie — immobilier, transport, PI, valeurs mobilières ; le bien reste à vous. 2. La plateforme émet l’AFN — sur la base de votre actif ; la Banque de Russie effectue l’émission ciblée. 3. Recevez des revenus — versements réguliers ; à l’expiration du terme, la charge est levée.

Toute étape peut être ignorée en tapant « Ignorer » dans le coin supérieur droit.

3.2. Écran d’accueil

L’écran d’accueil contient : - Carte de solde — valeur totale des actifs en activité ; nombre d’objets gagés ; revenu trimestriel. - Actions rapides : Soumettre un actif, Mes AFN, Revenus, Documents. - Bannière de projet national — nouveaux programmes de financement. - Mes actifs — les deux derniers actifs avec possibilité d’ouvrir la liste complète. - Dernières opérations — les deux dernières opérations avec possibilité d’ouvrir l’historique complet. - Navigation inférieure — cinq onglets : Accueil, Actifs, AFN, Revenus, Profil.

3.3. Soumission d’un actif en garantie

Étape 1. Sélection du type de bien

Catégories disponibles : - Immobilier (appartement, maison, apparts) ; - Parcelle de terrain ; - Véhicule ; - Propriété intellectuelle ; - Valeurs mobilières.

Étape 2. Caractéristiques

Pour l’immobilier — saisissez le numéro cadastral. L’adresse, la superficie, l’année de construction et d’autres caractéristiques seront chargées automatiquement depuis EGRN via un canal sécurisé d’intégration avec Rosreestr (FZ-218). Un titre de propriété est également joint.

Étape 3. Demande d’évaluation

Consultez les paramètres de la demande et confirmez l’envoi. Une société d’évaluation partenaire agréée selon FZ-135 effectuera l’évaluation sous 24 heures. Coût de la demande — 0 RUB (inclus dans le service plateforme).

Étape 4. Résultat de l’évaluation

Recevez la valeur d’évaluation et la taille maximale de l’AFN gagé (par défaut 50 % de l’évaluation). L’assurance du gage est incluse. Validité de 6 mois. Passez au contrat intelligent pour émettre l’AFN.

3.4. Configuration et signature du contrat intelligent

Sur l’écran du contrat intelligent, définissez : - Durée de l’AFN : 12, 36 ou 60 mois. - Utilisation ciblée : programme d’un projet national ou régional. - Valeur nominale de l’AFN : calculée automatiquement comme 50 % de la valeur d’évaluation.

Consultez les paramètres calculés : taux de rémunération, revenus pour toute la durée, agent escrow (Banque de Russie), assurance, registraire de la charge (Rosreestr).

Tapez « Signer (UKEP) ». Un code SMS sera envoyé à votre ligne téléphonique. Saisissez le code à 6 chiffres ou confirmez par la biométrie.

Après une signature réussie, vous verrez l’écran « AFN émis avec succès » avec un identifiant unique de l’AFN. L’AFN a été transféré à la Banque de Russie, la charge est enregistrée dans EGRN.

3.5. Gestion du portefeuille AFN

Dans la section « Mes AFN », la liste complète de vos AFN est disponible avec filtrage : Tous, Actifs, En émission, Terminés. Pour chaque AFN, est affiché : - ID (identifiant unique) ; - valeur nominale ; - actif lié ; - programme cible ; - durée et taux de rémunération ; - statut.

Un clic sur un AFN ouvre une fiche avec des informations détaillées : conditions du contrat intelligent, escrow, versements, historique. Actions disponibles : télécharger le contrat, demander un échange d’AFN.

3.6. Revenus

Dans la section « Revenus » : - Carte de solde des revenus : par an, par trimestre, prévision annuelle. - Graphique des recettes mensuelles. - Liste des versements liés à un AFN.

3.7. Historique des opérations

Journal complet des opérations avec filtrage : Tous, Versements, Émission d’AFN, Échange. Peut être exporté en PDF ou Excel.

3.8. Notifications

Le centre de notifications contient : - Messages système sur les versements ; - Notifications sur les nouveaux programmes ; - Notifications sur les résultats des évaluations ; - Notifications sur l’enregistrement des charges et d’autres événements du contrat intelligent.

Les notifications non lues sont signalées par un point d’accent. En tapant sur la cloche, on accède à toutes les notifications. En tapant sur l’icône « ⊘ », toutes les notifications sont marquées comme lues.

3.9. Profil

Dans la section « Profil » sont disponibles : - Données personnelles. - Coordonnées de versement. - Mes documents. - Notifications. - Documents de la plateforme. - Aide et support. - Déconnexion du compte.

3.10. Paramètres

Disponibles : - Langue : russe (par défaut), anglais, français, arabe. - Thème : clair, sombre, doux, contrasté pour les malvoyants. - Taille de police : plus petite, normale, plus grande. - Sécurité : biométrie, code PIN, authentification à deux facteurs. - Notifications : push-notifications.

3.11. Documents de la plateforme

La section contient des liens vers la documentation complète du projet CPFSR : hub des documents, grande présentation, petite présentation, site web, fondement juridique, évaluation internationale, audit DD, conclusion d’expert.

3.12. Support

Disponibles : FAQ, chat avec l’opérateur, téléphone de support, formulaire de retour.

4. Messages à l’opérateur

4.1. Types de messages

  • Push-notifications — affichées sur l’écran d’accueil de l’appareil.
  • Notifications in-app — affichées dans le centre de notifications.
  • Messages de Gosuslugi — affichés dans le service standard de Gosuslugi.

4.2. Actions sur les messages

Chaque notification contient un appel à l’action (par ex. « Confirmer le versement », « Lancer l’émission d’AFN »). Tapez la notification pour ouvrir l’écran correspondant.

5. Messages d’erreur

Code Message Action utilisateur
AUTH-01 Impossible de confirmer l’identité via Gosuslugi Réessayez, vérifiez l’accès à Gosuslugi
AUTH-02 Biométrie non confirmée Saisissez le code de confirmation ou confirmez via EBS
ASSET-01 Objet non trouvé dans EGRN Vérifiez l’exactitude du numéro cadastral
ASSET-02 L’objet est déjà gagé auprès d’une autre organisation Levez la charge précédente
CFA-01 Impossible d’émettre l’AFN Contactez le support
NET-01 Pas de connexion au serveur Vérifiez Internet ; les données seront restaurées après le rétablissement de la connexion
SIGN-01 Signature électronique rejetée Vérifiez la validité du certificat
