## Forme de livraison de la documentation

Tous les documents sont livrés aux formats :
- **Markdown (.md)** — source ;
- **HTML** — dans le style unifié du projet avec top-bar, thèmes, langues ;
- **DOCX** — pour édition dans Microsoft Word ;
- **PDF** — pour impression et archive.

La conversion est effectuée via pandoc avec le modèle commun `_shared/doc-template.html`.
