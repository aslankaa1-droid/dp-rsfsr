Matériels promotionnels pour publication dans RuStore

Composition du pack

1. Icônes de l’application

Objet Taille Fichier Format
Icône dans le catalogue 512 × 512 px icons/store-512.png PNG
Icône de l’application (adaptive) 192 × 192 px icons/icon-192.svg SVG
Icône de l’application (complète) 512 × 512 px icons/icon-512.svg SVG

Les icônes sont générées à partir du logo SVG source Приложение/icons/icon-512.svg.

2. Feature graphic / bannière

Objet Taille Description
Feature graphic 1024 × 500 px Bannière principale du catalogue ; comprend le logo, le nom, la phrase clé

3. Captures d’écran

Exigences minimales RuStore

  • Minimum 5 captures d’écran, maximum 8.
  • Ratio d’aspect : 16:9 (horizontal) ou 9:16 (vertical).
  • Taille minimale : 320 × 480.
  • Taille recommandée : 1080 × 1920 (vertical).
  • Format : PNG, JPEG.

Ensemble recommandé de captures d’écran (par flux de l’application)

№ Écran Capture Légende
1 Accueil screen-01-home.png « Gérez vos actifs dans une seule application »
2 Soumission d’actif screen-02-new-asset.png « Mettez le bien en garantie — le bien vous reste acquis »
3 Évaluation screen-03-valuation.png « Obtenez une évaluation professionnelle en 24 heures »
4 Contrat intelligent screen-04-contract.png « Configurez les conditions et signez le contrat intelligent »
5 Signature UKEP screen-05-sign.png « Confirmation par signature électronique qualifiée renforcée »
6 Catalogue AFN screen-06-cfa.png « Contrôlez votre portefeuille AFN »
7 Revenus screen-07-income.png « Versements réguliers de rémunération »
8 Profil screen-08-profile.png « Contrôle complet de la sécurité et des données »

4. Vidéo promo

Paramètre Valeur
Durée 15–30 secondes
Format MP4 (H.264 + AAC)
Résolution 1080 × 1920 (vertical)
Audio avec voix off ou sous-titres

Scénario (15 secondes) : - 0–3 s : logo + nom. - 3–6 s : accueil — solde et actifs. - 6–9 s : soumission d’actif. - 9–12 s : signature du contrat intelligent. - 12–15 s : « AFN émis » + slogan.

5. Textes

Description brève (80 caractères)

« Plateforme d’émission d’AFN contre garantie immobilière — service d’information-démonstration »

Description complète (voir le fichier 01_Описание_приложения.md)

Quoi de neuf

« Première release. Cycle de vie complet de l’émission d’AFN : de la soumission d’actif à la perception de revenus. »

Catégorie d’âge

18+ (application financière requérant un compte Gosuslugi confirmé).

Catégories

  • Principale : Finance.
  • Secondaire : Business.

Mots-clés

AFN, gage, immobilier, investissement, rouble numérique, Banque de Russie, projets nationaux

Instructions pour la préparation des captures d’écran

  1. Ouvrez l’Application dans un navigateur en mode simulation de téléphone (DevTools → Mobile, résolution 1080 × 1920).
  2. Naviguez vers l’écran correspondant.
  3. Prenez une capture via DevTools (Cmd/Ctrl + Shift + P → « Capture screenshot »).
  4. Si nécessaire — ajoutez légende et habillage dans un éditeur graphique.

Commande de génération des icônes (requiert ImageMagick ou Pillow)

# Avec Pillow / Python
python -c "
from PIL import Image
import cairosvg, io

for size in [44, 71, 96, 150, 192, 310, 512]:
    cairosvg.svg2png(url='Приложение/icons/icon-512.svg', write_to=f'store-{size}.png', output_width=size, output_height=size)
"

Alternativement — la CLI @capacitor/assets génère automatiquement toutes les tailles.
