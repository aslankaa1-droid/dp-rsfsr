Form of documentation delivery

All documents are delivered in the formats: - Markdown (.md) — source; - HTML — in the unified project style with top-bar, themes, languages; - DOCX — for editing in Microsoft Word; - PDF — for printing and archive.

Conversion is performed via pandoc using the common template _shared/doc-template.html.

The document complies with the requirements of GOST 19.507-79 “Inventory of operational documents”.
