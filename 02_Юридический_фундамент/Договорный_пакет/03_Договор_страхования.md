# Договор страхования ЦФА и Предмета залога (рамочный проект)

**Предмет:** страхование риска утраты, повреждения, гибели Предмета залога и риска ответственности оператора платформы.
**Правовая база:** Закон РФ от 27.11.1992 №4015-1 «Об организации страхового дела в Российской Федерации»; глава 48 «Страхование» Гражданского кодекса РФ; ФЗ от 25.04.2002 №40-ФЗ (применим для транспортных средств).

## 1. Стороны

1.1. **Страхователь** — АО «ЦП РСФСР» (Оператор) или Залогодатель (по схеме).

1.2. **Страховщик** — страховая организация, имеющая лицензию Банка России на осуществление имущественного страхования и/или страхования финансовых рисков (по виду страхования — на основании Указания ЦБ).

1.3. **Выгодоприобретатель** — держатель ЦФА (через расчётный механизм Оператора) при наступлении страхового случая в части предмета залога.

## 2. Объект страхования

2.1. **Предмет залога** — имущественные интересы, связанные с владением, пользованием и распоряжением Предметом залога.

2.2. **Ответственность Оператора** — имущественные интересы Оператора, связанные с обязанностью возместить причинённый им вред (страхование профессиональной ответственности оператора финансовой инфраструктуры).

## 3. Страховые случаи

3.1. Для Предмета залога — недвижимость:
- пожар, удар молнии, взрыв газа;
- стихийные бедствия (наводнение, землетрясение, ураган);
- противоправные действия третьих лиц (кража, поджог, грабёж);
- залив водой из соседних помещений;
- падение летательных аппаратов / их частей.

3.2. Для Предмета залога — транспортное средство: КАСКО-стандарт (угон + ущерб).

3.3. Для Предмета залога — объекты интеллектуальной собственности: страхование риска утраты исключительного права (при наличии бизнес-кейса).

3.4. Для ответственности Оператора:
- ошибки и упущения сотрудников Оператора, повлёкшие убытки держателей ЦФА;
- сбои информационной системы (cyber liability — в пределах согласованного лимита);
- утечка персональных данных и компенсации по ФЗ №152-ФЗ.

## 4. Страховая сумма и страховая премия

4.1. Страховая сумма по Предмету залога — не менее 100% его оценочной стоимости.

4.2. Страховая сумма по ответственности Оператора — не менее 500 миллионов рублей (по аналогии с минимальным лимитом для оператора инвестиционной платформы по 259-ФЗ).

4.3. Страховая премия рассчитывается по тарифной сетке Страховщика с учётом класса имущества, региона, дисконта по системе бонус-малус.

## 5. Срок страхования

5.1. Срок страхования совпадает со сроком обращения соответствующего ЦФА с учётом 30-дневного «хвоста» после погашения.

## 6. Порядок выплат

6.1. Страховое возмещение выплачивается Выгодоприобретателю в течение 30 дней с момента признания случая страховым.

6.2. В отношении ЦФА — выплата идёт на специальный распределительный счёт Оператора с последующим распределением по реестру держателей.

## 7. Исключения из страхового покрытия

7.1. Стандартные исключения по гл. 48 ГК + умысел Страхователя.

---

**Статус:** рамочный проект, подлежит адаптации под конкретного страховщика, конкретный объект и пакет рисков.
