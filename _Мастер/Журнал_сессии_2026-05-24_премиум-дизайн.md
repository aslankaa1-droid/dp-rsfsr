### 1.2 Дизайн-система — папка `_design-system/`

**Стили (CSS):**
- `tokens.css` — 3 темы (Light · Dark · Sepia) + шрифты, цвета, отступы, тени, моушн, focus-ring, градиенты
- `global.css` — reset + типографика (Inter Tight / Inter / JetBrains Mono / IBM Plex Sans Arabic) + таблицы + кастомный scrollbar + анимации
- `components.css` — top-bar sticky с blur, ticker, dropdown menus, drawer mobile, buttons (primary/gold/ghost), cards, badges, hero, KPI-strip, footer, back-to-top, breadcrumbs, skip-link, modal
- `rtl.css` — поддержка арабского (RTL)
- `print.css` — печатные стили для PDF-экспорта с колонтитулами

**Скрипты (JS):**
- `theme-switcher.js` — переключатель 3 тем + системная автодетекция через `prefers-color-scheme` + persistence в localStorage
- `i18n.js` — переключатель языков, авто-старт по `navigator.language`, RTL автодетект, fallback на RU
- `ticker.js` — бегущая строка с 15 котировками (USD/EUR/CNY/AED против ₽, золото, серебро, платина, палладий, Brent, WTI, газ, медь, никель, алюминий, ставка ЦБ). Snapshot от 24.05.2026 + live-обогащение через `open.er-api.com` (без ключа) + кеш TTL 60 мин в localStorage
- `ui.js` — sticky top-bar shadow, back-to-top, dropdown-меню, drawer, reveal-on-scroll через IntersectionObserver

**Локализация (3 языка):**
- `i18n/ru.json`, `i18n/en.json`, `i18n/ar.json` — по ~70 ключей в каждом (brand, nav, actions, themes, hero, kpi, sections, footer, ticker)

**Логотип и идентика:**
- `logo.svg` — основной (щит-обрамление в emerald-градиенте, золотая колонна-устой, три горизонтальные ступени = Stage 1 / Stage 2 / Pilot, акцент-точка Pilot Light wedge)
- `logo-mono.svg` — монохромная версия (`currentColor`) для рамок/печати
- `logo-lockup.svg` — горизонтальный лок-ап со знаком и текстовой подписью
- `favicon.svg` — иконка вкладки

**Партиалы:**
- `partials.html` — справочник готовых блоков (`<head>`, top-bar, ticker, drawer, footer, back-to-top) для копипасты в новые HTML
