---

## 1. Что было

При повторной проверке выявлено: главная (`index.html`) и карта сайта (`site-map.html`) имели премиум-топ-бар из новой дизайн-системы, но при переходе на любой документ — топ-бар был **другой** (старый из Горизонта III, на `_shared/brand.css`). Причина: pandoc-шаблон `_shared/doc-template.html` всё ещё использовал старый top-bar образца Горизонта III с собственным набором стилей.

Затронуто **26 HTML-документов**, собранных через тот шаблон. На каждом из них при переходе с главной топ-бар «прыгал» в визуальном дизайне.

---

## 2. Что сделано

### 2.1 Переписан `_shared/doc-template.html`

Старый шаблон (data-theme="daylight", `_shared/brand.css`, свой top-bar с RU/EN/FR/AR/A11Y кнопками) заменён на единый шаблон, который:

- подключает `_design-system/{tokens,global,components,rtl,print}.css`
- подключает `_design-system/{theme-switcher,i18n,ticker,ui}.js`
- содержит **точно тот же** sticky-топ-бар, что и `index.html` (логотип-щит + 6-секционная навигация + переключатели языка / темы / бургер)
- содержит **ту же бегущую строку котировок** (sticky под топ-баром)
- содержит **тот же drawer** для мобильной навигации
- содержит **тот же footer** с rights + engagement reference
- содержит **back-to-top** кнопку
- содержит **breadcrumb** (Главная / Карта сайта / Категория) для контекста
- сохраняет document-specific layout: doc-wrap (920 px max-width), doc-header с eyebrow + h1 + meta, doc-content с премиум-типографикой (Inter / Inter Tight / JetBrains Mono), таблицы с tabular-nums и hover-эффектом, цитаты, code-блоки, hr, кнопки «На главную» / «Карта сайта» в конце
- `cpfsr-root` meta — пересчитывается из `$shared-prefix$` (для корректного нахождения i18n-словарей из любой глубины папок)

### 2.3 Mermaid-диаграммы

`01_Техническая_документация/08_Архитектурные_диаграммы.html` содержит 7 mermaid-блоков. Pandoc post-process в `build_html.sh` сломался на кириллических путях (Python кодировка). Сделан альтернативный post-process через PowerShell с UTF-8 байтами напрямую: pandoc-обёртки `<pre class="sourceCode mermaid">` заменены на `<pre class="mermaid">`, HTML-entities декодированы, подключён `mermaid@10` через ESM-импорт. Все 7 диаграмм отрисовываются.

### 2.4 Что НЕ трогалось (по приказу «не придумывай ничего нового»)

- `index.html` и `site-map.html` — уже были на новом топ-баре, не нужно
