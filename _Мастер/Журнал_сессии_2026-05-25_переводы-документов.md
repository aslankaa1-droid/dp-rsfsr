## 1. Честный масштаб задачи

В проекте — **70 русскоязычных MD-документов** общим объёмом ~1 МБ исходного текста. Перевод на 3 языка = **210 переводных документов**, ~3 МБ премиум-перевода.

Физически в одну сессию это не помещается (контекстный лимит). Я выбрал стратегию **честно сделать максимум за сессию** и оставить подробный план на продолжение, а не имитировать «всё готово».

## 2. Сделано в этой сессии

### 2.1. Инфраструктура переводов

### 2.3. Качество переводов

**EN** — институциональный регистр Series A roadshow:
- ЦФА → DFA (Digital Financial Assets)
- 779-P, 4336-U сохранены латиницей (стандарт DD-пакетов)
- «государственный по природе, частный по эффективности» → «sovereign in mandate, private in execution»
- «целевое фондирование через инфраструктурный контур» → «targeted funding through the institutional infrastructure of VEB.RF / Federal Treasury»

**FR** — pour Paris / Genève / Luxembourg / Casablanca / Montréal:
- TRI / VAN / CMPC (финансовые сокращения)
- «Pacte d'actionnaires», «Conseil d'administration», «Comité d'audit» — стандартная французская корпоративная лексика
- Apostrophes typographiques, accent правильные

**AR** — высокий regular الفصحى для GCC / KSA / UAE / Qatar:
- معدل العائد الداخلي (IRR), صافي القيمة الحالية (NPV)
- بنك روسيا, البنك المركزي الروسي (Bank of Russia)
- المعرفة الفنية (know-how), شهادة الإيداع (deposit certificate)
- Латиница сохранена для VEB.RF, Gazprombank, Atomyze (стандарт MENA финансовой прессы)
- RTL — автоматический

## 5. Как проверить работу прямо сейчас

1. Открыть `_Мастер/Концепт.html` (или `17_Data_room/INDEX.html`, или `16_Governance_и_Compliance/Governance_Pack.html`)
2. В шапке справа — нажать флаг 🇬🇧 / 🇫🇷 / 🇸🇦
3. Откроется ПОЛНОСТЬЮ переведённая версия документа (не только шапка)
4. Обратно на русский — флаг 🇷🇺
5. Для арабского — текст автоматически RTL

Для всех остальных документов сейчас (где перевод ещё не сделан) переключатель языка переводит только шапку, тикер, навигацию (UI), а тело документа остаётся на русском — потому что реестр честно знает, что эти 67 документов ещё не переведены, и не пытается их редиректить.
